**Цифровое портфолио**
-
Данный проект сделан для общего представления задуманного функционала


**Идея проекта**
-
Реализация цифрового портфолио для студентов, которое будет:
- Сопровождать студентов во время обучения и во время их выпуска
- Ранжировать студентов для поиска нужных людей
- Состовлять рейтинговый список студентов
